# AK-47 Recoil Control Macro

Ця програма є інструментом для компенсації віддачі (recoil) зброї AK-47, адаптованим для шутерів, таких як CS2 або Rust. Основна мета проєкту — забезпечити стабільний рух курсора з правильними таймінгами та зручним графічним інтерфейсом, оскільки більшість публічних скриптів страждають від розсинхронізації під час тривалого затиску (спрею).

## Основні можливості

- **Графічний інтерфейс:** Створений за допомогою бібліотеки customtkinter.
- **Точні таймінги:** Стандартна функція time.sleep() у Windows має суттєву похибку. У цьому коді використовується процесорний таймер (perf_counter), що дозволяє тримати темп стрільби рівно 600 пострілів на хвилину без відхилень.
- **Плавність рухів:** Рух між пострілами розбивається на задану кількість мікрокроків. Це робить компенсацію плавною та запобігає ривкам курсора.
- **Масштабування чутливості:** Амплітуда рухів автоматично перераховується залежно від вказаної в налаштуваннях чутливості (sensitivity) миші.
- **Візуалізація:** Програма має вбудоване полотно для відображення траєкторії патерну.

Редагування патерну
Якщо базовий патерн потребує коригування під ваші потреби, ви можете змінити його безпосередньо в коді.
У файлі main.py знаходиться масив _RAW_PATTERN. Кожен елемент масиву — це координати (X, Y) для окремої кулі:
Додатній X — рух вправо, від'ємний X — рух вліво.
Додатній Y — рух вниз, від'ємний Y — рух вгору.
Коригуйте ці значення на кілька пунктів для досягнення ідеальної точності.
